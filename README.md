# MFEA-Dataset
针对现有面向工业产品的加工特征识别数据集在特征类型单一、几何复杂度不足、与真实工业零件差距较大以及语义层级划分粗糙等方面的局限性，构建了一种面向机车转向架等工业产品的三维模型特征面识别的工业级语义分割数据集，用于结构件加工特征的语义解析与点云数据的语义判别。数据集设计了多层级语义标签划分策略，包含了强调“结构相似但语义不同”、强调“语义一致但结构不同”的标签体系，用于提升系统在机车转向架等复杂工业产品存在的语义混淆与结构多变问题下的鲁棒性与泛化能力。以工业领域典型零件齿轮类为试验案例，基于真实设计公式进行参数生成与数据集扩展，同时引入轮毂、辐板、减重孔等真实附加结构，增强了模型的几何耦合性与拓扑复杂性，并通过设计参数—几何构造—拓扑单元之间的可追溯映射关系实现多层级语义标签的自动标注。
